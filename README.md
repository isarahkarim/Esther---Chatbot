# Esther-Chatbot
Esther is a Python chatbot that blends structured dialogue, interactive mini-games, and Wikipedia-powered knowledge lookup. The project focuses on interaction design and clean Python structure.

# Features

- Conversational Interaction: Engages users with text-based dialogue using rule-based logic.
- Mini Games: Includes small interactive text-based games for fun and practice.
- Wikipedia Lookup: Fetches summarized information from Wikipedia on a wide range of topics.
- Modular Design: Code is structured to allow easy expansion with new features.
- Expandable Logic: Built with the potential for memory, personalization, and improved responses.
